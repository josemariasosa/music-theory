## 4. Harmonizing Major scales.

### Basic

From the bass perspective, going up one fret it is called "half-step". This is because to go from C to D two half-steps are reaquired, aka. a step. If we start from C and use the following pattern of half and whole steps we can build any major scale. In music you can also call them tone and semitone.

```text
Major Scale:

┣━━━━━━╋━━━━━━╋━━━╋━━━━━━╋━━━━━━╋━━━━━━╋━━━╋
1      2      3   4      5      6      7   8
  tone   tone       tone   tone   tone
             semitone                 semitone
```

The sound of 1 and 8 are almost the same. They resonate! This mean that we have move an octave from the original tone. The other seven more tones build what we know as the Major scale.

So, to move from 1 to 2, we have to move a tone. To go from 1 to 3, we have to move 2 tones. To 1 to 3, we have to move a **major third**, which mean 1.5 tones.

### Stacking thirds

Stacking thirds is a process we use to build, "to harmonize", all the chords from the major scale.

```text
Major Scale:

┣━━━━━━╋━━━━━━╋━━━╋━━━━━━╋━━━━━━╋━━━━━━╋━━━╋
C      Dm     Em  F      G      Am     Bm° C
I      ii     iii IV     V      vi     vii°
1      2      3   4      5      6      7   8
  tone   tone       tone   tone   tone
             semitone                 semitone
```

#### 3 notes chords

To build all the 3 notes chords from the major scale we need to remember from the **chromatic intervals table** that a major 3rd equals to 2 tones, and the minor 3rd equals to 1.5 tones. 

All the major chords are build using the following formula:

```text
chord = root + major 3rd + minor 3rd
```

For example, if our root is C, the C major chord is build using the notes: C + E + G.

On the other hand, all the minor chords are build using the following formula:

```text
chord = root + minor 3rd + major 3rd
```

For example, if our root is A, the A minor chord is build using the notes: A + C + E.

If you did the math, you could notice that, if we start with a root of value 0 and then we sum the major 3rd tones and the minor 3rd tones, either way, we obtain 3.5 tones. If we go back again to the chromatic intervals table, we can see that this measure it's called **Perfect Fifth**. 

It is called perfect fifth because it does not matter if the chord is major or minor, the root and the fifth stay the same. The only one that changes is the third, which could be major or minor 3rd.

We have only stack major along with minor thirds, but what would happen if we stack 2 major or 2 minor thirds. The answer is, we obtain the other 2 types of 3 notes chords that are: diminished and augmented chords. The latter being the less popular of the four chords, since the augmented chord are not part of the major scale.

As a summary

```text
|---major-----|--minor---|    -> major chord

|--minor---|---major-----|    -> minor chord

|--minor---|--minor---|       -> diminished chord

|---major-----|---major-----| -> augmented chord
```

Notice how, for diminished chords, the fifth is not perfect. A minor 3rd (1.5 tones) plus a minor 3rd (1.5 tones) give us a total of 3 tones. This interval receive the name of **diminished fifth**.

On the other hand, a major 3rd (2 tones) plus a major 3rd (2 tones) give us a total of 4 tones. This interval is called, try to guess how?, **augmented fifth**. Sadly, this chord is not present in any major scale, however it could be found in other scales like the Melodic Minor Scale.

##### C major chord (grade I)

Let's talk about C major as a chord. If we start from 1, from C, and we move a **major third**, 2 tones, we land on E. Then, if from 3, we move another third, a minor third, then we reach G, the note in the fifth grade.See what we are doing here? Stacking thirds.

This tell us that the C major, or simply C, chord is generated by using all those notes: C, E, G.

```text
Major Scale:

|---1 third---|--2 third-|
|             |          |
|---major-----|--minor---|
*             *          *
┣━━━━━━╋━━━━━━╋━━━╋━━━━━━╋━━━━━━╋━━━━━━╋━━━╋
C      D      E   F      G      A      B   C
1      2      3   4      5      6      7   8
  tone   tone       tone   tone   tone
             semitone                 semitone
```

There could be many, but these are the most common chords for C using a 4 string bass.

```text
  |    C    |    C    |                                                       !
G |---------|----9----|-------------------------------------------------------|
D |----5----|---------|-------------------------------------------------------|

E |----8----|----8----|-------------------------------------------------------|
```

The first C is being play: C-E-G, and the second C: C-G-E. Technically the same, but aestetically not.

Walk them all, saying out loud the chord grade, the note, the chord, could be very helpful to remember the names. Try to memorize the shape of the left hand, the chord name and the grade.
